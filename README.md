# custom_paramter ROS-Package
This repository includes the custom_paramter package. This is a dependency i use in some of my other projects.

## Installing
Just clone in your ROS-Workspace and build it with catkin. 

### Prerequisites 
* ROS and ROSCPP :-)
* Boost (lexical_cast)
